## Interpretable CHD Risk Prediction

<img src="/images/projects/roc_curve.png" width="250" align="right" style="margin-left:25px; border-radius:10px;">

Developed an end-to-end statistical learning framework for predicting 10-year coronary heart disease risk using the Framingham Heart Study dataset. Additionally, I compared traditional and regularized logistic regression approaches to evaluate the tradeoff between predictive performance, model complexity, and interpretability.

The project emphasized reproducible statistical methodology through cross-validation, calibration analysis, bootstrap uncertainty estimation, and explainable AI techniques.

**Methods**

- Logistic regression
- LASSO regularization
- ROC curve analysis
- Feature selection
- SHAP interpretation

**Results**

- Accuracy: 84.4%
- ROC-AUC: 0.70
- Evaluated calibration and uncertainty using bootstrap confidence intervals

**Tools**

Python • Statistical Learning • Explainable AI

## Computational Biology & Biomedical Representation Learning

<img src="/images/projects/principal_component.png" width="250" align="right" style="margin-left:25px; border-radius:10px;">

Developed a self-supervised autoencoder pipeline to learn meaningful representations from high-dimensional molecular dynamics datasets related to steroid hormone receptor systems. This project investigates how machine learning can identify hidden biological patterns connecting small molecules, receptor structures, molecular conformations, and downstream signaling pathways. The broader goal is to improve drug discovery by understanding molecular mechanisms that influence therapeutic efficacy and off-target effects. The pipeline explores the statistical foundations underlying representation learning, including data normalization, decorrelation, clustering, dimensionality reduction, and optimization of model parameters.

**Methods**

- Self-supervised learning
- Autoencoder architectures
- Data normalization
- Principal component analysis
- Dimensionality reduction
- Clustering analysis
- Cross-validation
- Hyperparameter optimization

**Concepts**

Biomedical AI • Computational Biology • Representation Learning • Statistical Learning
