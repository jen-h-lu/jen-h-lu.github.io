# Projects

My projects explore statistical learning, machine learning, and computational approaches for extracting meaningful insights from complex datasets.

I am particularly interested in methods that balance predictive performance, interpretability, and reproducibility.

## Interpretable CHD Risk Prediction

<img src="/images/projects/roc_curve.png" width="250" align="right" style="margin-left:25px; border-radius:10px;">

Developed an interpretable machine learning pipeline for coronary heart disease risk prediction using the Framingham Heart Study dataset.

Rather than optimizing prediction alone, this project examines how statistical learning methods can provide transparent and clinically meaningful insights.

**Methods**

- Logistic regression
- LASSO regularization
- ROC curve analysis
- Feature selection
- SHAP interpretation

**Tools**

Python • Statistical Learning • Explainable AI
