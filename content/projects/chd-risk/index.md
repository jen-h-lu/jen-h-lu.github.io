# Interpretable CHD Risk Prediction

This project develops an interpretable statistical learning pipeline for predicting coronary heart disease risk using the Framingham Heart Study dataset.

## Methods

- Logistic regression
- LASSO regularization
- ROC-AUC evaluation
- SHAP interpretation
